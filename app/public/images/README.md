# Villager Images

Place villager images here with the following naming convention:

- `{shortName}_smile.png` - Smiling villager (when marked as truthful during gameplay)
- `{shortName}_angry.png` - Angry villager (when marked as werewolf/shill during gameplay)
- `{shortName}_werewolf.png` - Werewolf form (when solved and villager is a werewolf)
- `{shortName}_shill.png` - Shill form (when solved and villager is a shill)
- `{shortName}_releaved.png` - Relieved form (when solved and villager is truthful)

Where `{shortName}` is one of: Alice, Bob, Charlie, Doris, Edith, Frank

For now, placeholder images will be shown if images are not found.
